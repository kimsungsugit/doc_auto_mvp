# 공개 양식 기반 합성 샘플 메모

이 샘플 세트는 실제 고객 문서를 포함하지 않습니다. 아래 기준을 바탕으로 만든 합성 샘플입니다.

## 기준
- 국세청/홈택스 전자세금계산서 안내 문서에서 확인 가능한 필드 구조
- 공개적으로 볼 수 있는 세금계산서 양식의 정보 배치
- 실제 세무 문서에서 자주 나오는 공급자, 공급받는자, 작성일자, 금액, 품목, 비고 패턴

## 추가한 실문서형 케이스
- 박스형 양식
- 영수 표기형 전자세금계산서
- 수정 발급 사유가 붙은 케이스
- 하단 승인 문구가 들어간 케이스
- 스캔본처럼 배경 노이즈와 기울기를 준 케이스
- 거래명세서 스캔풍 케이스

## 목적
- 고객 실문서가 없을 때도 레이아웃 편차 테스트 범위를 넓히기 위함
- OCR 이전 단계에서도 시각적으로 실문서에 가까운 데모를 만들기 위함
- 마지막에 비식별 실제 PDF 2~3건이 들어왔을 때 튜닝 범위를 줄이기 위함
